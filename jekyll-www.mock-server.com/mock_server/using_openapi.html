MockServer supports OpenAPI v3 or Swagger v2 specifications in either JSON or YAML format.

These specification can be used in request matchers for

  • matching expectations
  • verifying requests and request sequences received
  • selectively clear logs
  • selectively retrieving logs, received request, expectations, etc

These specification can also be used to generate expectations with example responses.
